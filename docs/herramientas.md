# Lista de Herramientas - Suite Escuadra

Este documento describe las herramientas principales que forman parte del proyecto de la calculadora de ingeniería, detallando su propósito y estado actual.

## Herramientas de la Suite

| Nombre | Propósito | Estado |
| :--- | :--- | :--- |
| **Calculadora de Vigas** | Cálculo de momentos flectores y fuerzas cortantes en estructuras simples. | Pendiente |
| **Conversor de Unidades** | Herramienta para transformar medidas técnicas (presión, fuerza, longitud) al SI. | En progreso |
| **Análisis Estadístico** | Procesamiento de datos experimentales y cálculo de desviaciones estándar. | Pendiente |
| **Diseño de Mezclas** | Cálculo de proporciones para materiales de construcción según normativas. | Pendiente |

## Notas Adicionales
- Todas las herramientas deben cumplir con el estándar de precisión numérica establecido en el plan de pruebas.
- El desarrollo se basará en los requerimientos discutidos en clase.
